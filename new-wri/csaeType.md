# CaseType

## CaseType

|  類型   |        內容         |
|:-----:|:-----------------:|
| **0** |         無         |
| **1** | 無特殊內容(通常為VV的原先類型) |
| **2** |        字串         |
| **3** |        表單         |
| **4** |        函數         |
| **5** |       無點數字        |
| **6** |       有點數字        |
| **7** |       可重疊符號       |
| **8** |      不可重疊符號       |
